# SHL Recommender

AI-powered recommender system using Groq and ChromaDB.

## Structure
- `scraper.py`: Data ingestion from web sources.
- `build_index.py`: Vector database indexing with ChromaDB.
- `agent.py`: LLM agent logic using Groq.
- `main.py`: FastAPI application entry point.
- `eval.py`: Evaluation suite for recommendation quality.
